# Running

## Installing Dependencies
#####`npm install`

####For running Grunt tasks:
#####`npm install -g grunt-cli`

## Running Tests

#####`grunt test`
   - Lints, builds, and tests

## Serving

#####`grunt serve`
   - Lints, builds, and tests
   - Starts up a server and watches for changes
